# CSS选择器

## 页面包含CSS的方式

```
行间样式 => 直接在标签内引用
内嵌式 => 在header里的style标签下编写
链接式 => 在header里通过link标签导入
```



## CSS选择器

### 1.标记选择器（标签选择器）

```html
<a href=""></a>

a{
	font-size:25px;
}
```

### 2.类选择器class

```html
<a class="aaa" href=""></a>

.aaa{

}
```

### 3.id选择器

```html
<a id="name" href=""></a>

#name{

}
```
